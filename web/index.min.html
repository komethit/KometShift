Let's Start Research

A convenient code editor for editing files for Python, there is also a convenient system of packages and plugins for customizing the editor for yourself, or you can even create your own using the functions that are already included in applications from the box, there is also a convenient code launch system and an internal console.

Customizing Your Project

This application has a lot of settings for the special operation of the application, the first and probably the most important file is settings.json in which all the information of the application and also its composition, there are also many descriptions and variables such as displaying different parts of the code and so on, they can all be changed , there is no instruction for this, you see everything is simple and clear even looking at the names of these variables.

In this application, in addition to changing the settings, you can customize the color theme for yourself, for this you need to change the variables with colors, you can also create your own themes simply by making a new package and upload your theme, for this you just need to read about creating your package, you can also easily customize the font and put your.

Downloads

What do you need to install:
1. Download the .zip archive of the project from github and upload it to your working folder.
2. Start terminal and enter project folder: cd [path].
3. Run the setup file: python setup.py command.
I advise you to use other editors to run the application After installation, be sure to download the necessary modules (they are in the settings.json file) and there the "modules" list is a list of the necessary.

Keyboard Shortcuts

You can set up your hotkeys in a separate file where you can put your own keys, the main thing is that there is no such one or the system key is not occupied.

Privacy and Anonymity
